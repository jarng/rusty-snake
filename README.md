# rusty-snake

A simple snake game written in Rust for educational purposes.

![image](https://github.com/nhthieu/rusty-snake/assets/74890715/d4ede976-6c4e-429e-9ffa-116e00811139)

## Prerequisites:

- [Rust](https://www.rust-lang.org/tools/install)

## Quick start

- Run:
```bash
cargo run --release
```
